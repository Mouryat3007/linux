to add user
   > sudo useradd <username>

TO add a group
   > sudo groupadd <group name>

   groupadd= to add new group

TO add users to the group
   > sudo usermod -aG <groupname> <username>

   -aG= add group

To change group of a file
   > sudo chgrp < newgroupname> <filename> 

   chgrp= change group
